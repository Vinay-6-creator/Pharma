Clear solution example: dosage calculation

Yeh example patient ke weight aur dose rate se total dose calculate karta hai.

weight = float(input("Patient weight (kg): "))
dose_rate = float(input("Dose rate (mg/kg): "))
total_dose = weight * dose_rate
print(f"Total dose required: {total_dose} mg")
Solution explanation: agar patient ka weight 60 kg ho aur dose rate 5 mg/kg ho, output hoga 300 mg. Conversion float() zaroori hai taaki decimal values bhi correct handle ho.

Clear solution example: BMI calculation and category

def calculate_bmi(weight_kg, height_m):
    return weight_kg / (height_m ** 2)

weight = float(input("Weight (kg): "))
height = float(input("Height (m): "))

bmi = calculate_bmi(weight, height)
print(f"BMI: {bmi:.2f}")

if bmi < 18.5:
    category = "Underweight"
elif bmi < 25:
    category = "Normal"
elif bmi < 30:
    category = "Overweight"
else:
    category = "Obese"

print(f"Category: {category}")
Solution explanation: function calculate_bmi return kar rahi hai BMI value, phir conditions check karte hain patient ko category assign karne ke liye.

Clear solution example: drug dose dictionary and list operations

drugs = {
    "Paracetamol": 500,
    "Ibuprofen": 200,
    "Amoxicillin": 250
}

patient_name = input("Patient name: ")
selected = input("Choose drug: ")

if selected in drugs:
    dose = drugs[selected]
    print(f"{patient_name} should take {dose} mg of {selected}.")
else:
    print("Selected drug not available.")
Solution explanation: dictionary se drug ki dose read karenge. Agar drug available nahi hai, user ko message milega.

Clear solution example: read a CSV file with healthcare data

import csv

with open('pharma_data.csv', mode='r', newline='') as file:
    reader = csv.DictReader(file)
    for row in reader:
        print(row['PatientID'], row['Age'], row['Diagnosis'])
Solution explanation: CSV file ko DictReader se read karte hain, jisme column names keys ke roop mein milte hain.

Clear solution example: inspect and clean dataset

import pandas as pd

df = pd.read_csv('pharmacy_records.csv')
print(df.head())
print(df.info())
print(df.describe())

# Drop rows with missing dose values
clean_df = df.dropna(subset=['Dose'])

# Filter patients older than 60
elderly = clean_df[clean_df['Age'] >= 60]
print(elderly)
Solution explanation: dataset inspect karne ke baad missing values remove kiye, phir age filter apply kiya for elderly patients.

Clear solution example: group by drug type and count

grouped = clean_df.groupby('DrugName').size()

Solution explanation: groupby se drug wise patient count milta hai, jo clinical reporting aur inventory planning mein useful hai.

Clear solution example: plot drug concentration over time

import matplotlib.pyplot as plt

time = [0, 1, 2, 4, 6, 8]
concentration = [0, 8.5, 15.2, 20.3, 14.8, 8.2]

plt.plot(time, concentration, marker='o', color='#2563eb')
plt.title('Drug Concentration vs Time')
plt.xlabel('Time (hours)')
plt.ylabel('Concentration (mg/L)')
plt.grid(True)
plt.show()
Solution explanation: line plot se drug absorption aur elimination phase clearly nazar aata hai. Axes labels aur title report ko readable banate hain.
